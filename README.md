# JavaScript Foundation Projects

A collection of JavaScript mini projects built to strengthen programming fundamentals, logical thinking, and real-world problem solving using arrays, objects, functions, and data manipulation techniques.

## Projects Included

### 1. Shopping Cart Analysis
A mini shopping cart system that stores supermarket items using arrays and objects, calculates totals, and prints formatted receipts using template strings.

#### Features
- Item storage using objects
- Quantity and price tracking
- Receipt generation
- Total price calculation
- Array manipulation

---

### 2. Library Book Tracker
A simple library management simulation system using arrays and objects.

#### Features
- Book check-in/check-out system
- Due date tracking
- Availability status
- Dynamic library updates
- Object-based data handling
